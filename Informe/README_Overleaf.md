# Informe para Overleaf

Archivo principal recomendado para el informe: `main.tex`.

Archivo principal recomendado para la presentacion: `presentacion.tex`.

Tambien se puede compilar `Template.tex`, que solo redirige a `main.tex`.

Antes de entregar, editar en `sections/00_portada.tex`:

- Nombre y codigo de los 3 integrantes.
- Nombre del docente, si corresponde.

Carpetas incluidas:

- `sections/`: cuerpo del informe separado por secciones.
- `tables/`: tablas LaTeX reutilizables.
- `ImgNod1/` e `ImgMultiNodo/`: capturas usadas como evidencia.
- `codigo/mdjm/`: archivos Scala adjuntos.
- `salidas/`: salidas JSON livianas de consultas, metricas y curvas de perdida.

No se copio el CSV completo ni las tablas JSON completas de `output/tables/` para no hacer pesado el proyecto de Overleaf.
